👷 안전보건관리체계 평가 시스템 (Safety & Health Management System Assessment)1. 프로젝트 개요본 프로젝트는 기업의 안전보건관리체계 구축 및 이행 현황을 정밀하게 확인하고 평가하기 위한 웹 기반 시스템입니다. 사용자가 "안전보건관리체계 평가표" 기준에 따라 평가를 진행하고, 첨부된 증빙 자료를 AI가 분석하여 평가의 신뢰성을 높이며, 누적된 평가 데이터를 기반으로 업체별 안전보건 수준의 변화 추이를 시각적으로 비교 분석하는 것을 목표로 합니다.2. 주요 기능 및 특징2.1. 평가 시스템 핵심 기능기능 영역주요 내용비고평가 진행평가표 항목별 점수 입력 및 환산 (총점 100점 만점)항목별 배점은 첨부된 평가표 기준첨부 파일 관리각 평가 항목별로 증빙 자료(사진, PDF) 첨부 가능다수의 파일 첨부 지원코멘트 및 메모각 항목에 대한 평가자 코멘트 및 AI 코멘트 추가 기능사용자가 AI 코멘트를 수정 가능업체별 평가 이력업체별 모든 평가 기록을 DB에 저장 및 누적 관리평가 비교 분석다회 평가 업체에 대해 이전 평가 결과와 현재 평가를 비교하는 화면 제공평가 시점마다 비교 분석 항목 추가 여부 선택 가능2.2. AI 기반 분석 및 평가 지원증빙 자료 분석: 첨부된 사진 또는 PDF 파일의 내용을 분석하여 해당 평가 항목의 '이행 내용'과의 일치 여부를 판단하고 적정성을 평가합니다.자동 코멘트 생성: AI 분석 결과를 바탕으로 평가 항목에 대한 객관적인 코멘트를 자동으로 생성하여 평가자의 업무를 지원합니다.총평 작성 지원: 최종 취합된 모든 평가 데이터를 AI가 종합적으로 분석하여 안전보건관리체계 시스템의 총평을 작성하며, 사용자는 이 초안을 수정하여 최종 보고서에 활용할 수 있습니다.AI 모델 설정: 관리자 메뉴에서 GPT, GEMINI 등의 API 모델을 선택하고, 각 기능(자료 분석, 총평 작성 등)에 사용할 프롬프트를 직접 설정할 수 있습니다.2.3. 보고서 생성 및 관리PDF 보고서 생성: 최종 평가 결과를 포함하는 전문적인 PDF 보고서를 생성합니다.보고서 저장 및 미리 보기: 생성된 보고서를 저장하고 인쇄 전 미리 보기 기능을 제공합니다.표지 디자인 관리:최종 보고서 출력 시, 표지 디자인 파일(pdf, jpg 등)을 업로드할 수 있습니다.관리자가 사전에 등록/관리하는 디자인 파일을 선택하여 적용할 수 있습니다.보고서 메타데이터 설정: 보고서 제목, 소제목, 업체명, 날짜 등을 직접 입력할 수 있으며, 업체명과 오늘 날짜가 기본값으로 자동 입력됩니다.3. 기술 스택 및 개발 구조구분기술 스택설명프론트엔드Next.js (React)성능 최적화 및 서버 사이드 렌더링 지원스타일링Tailwind CSS빠르고 일관된 디자인 시스템 구축 (모바일 반응형 지원)백엔드/APINext.js API Routes안전한 API 엔드포인트 구축데이터베이스PostgreSQL (Supabase 연동)평가 기록, 업체 정보, 회원 정보 등 데이터 저장AI 연동Google Gemini API 등첨부 파일 분석 및 총평 작성 지원인증Supabase Auth사용자 인증 및 관리자/일반 회원 권한 분리4. UI/UX 및 디자인 가이드디자인 프레임워크: Tailwind CSS글꼴: Pretendard 폰트 기본 적용테마: 다크 모드(Dark Mode) 및 라이트 모드(Light Mode) 기본 지원아이콘: 가독성을 높이기 위해 최대한 흑백(Gray-scale) 아이콘 사용을 지향합니다.5. 메뉴 및 페이지 구성5.1. 회원 로그인 페이지사용자 유형접근 권한필드관리자모든 관리 기능 접근아이디, 비밀번호, 이름, 휴대전화, 이메일, 회사일반 회원평가, 업체 관리 등 일반 기능 접근아이디, 비밀번호, 이름, 휴대전화, 이메일, 회사5.2. 일반 회원 메뉴평가 업체 (Client List):업체 리스트 조회 및 검색 (업체명, 대표자명, 업종 등)각 업체의 최종 평가 점수 및 평가 그래프, 총평 요약 제공평가하기 버튼 (기존 평가 내용 기반 또는 신규 평가 시작)업체 추가하기 (Add Client): 신규 평가 업체의 기본 정보 입력설정 (Settings): 사용자 계정 정보 수정, 테마 설정 (다크/라이트 모드)5.3. 관리자 메뉴 (Admin Console)회원 설정 (User Management):회원 목록 조회, 가입 승인/거절, 삭제, 정보 수정, 관리자 권한 부여평가 업체 관리 (Client Management):업체 정보 추가/수정/삭제업체별 모든 평가 내용 확인 및 수정 권한 관리평가 그래프 설정 (Chart Settings):대시보드 및 보고서에 사용될 평가 그래프 유형 선택 (원형, 방사형, 막대 그래프 등)AI 설정 (AI Configuration):사용할 AI API 모델 선택 및 설정 (GPT, GEMINI 등)평가 분석 및 총평 작성에 사용될 프롬프트 템플릿 설정 및 관리인쇄 설정 (Print Settings):인쇄 표지 디자인 및 폼 디자인 수정 관리: 보고서 표지 및 본문 폼의 템플릿 관리페이지 표시 설정: 평가표에 맞는 머리말(Header), 바닥글(Footer), 페이지 번호 표시 설정5.4. 대시보드 (Dashboard)검색: 업체명, 업종 등으로 업체 리스트 검색 기능업체 리스트:업체명대표자명, 업종평가 점수 그래프 (설정된 그래프 유형으로 시각화)업체 총평 (AI가 작성하고 사용자가 수정한 최종 총평 요약)평가하기 버튼 (새 평가 시작 또는 기존 평가 수정)6. 데이터베이스 구조 (Supabase/PostgreSQL)테이블 명설명주요 필드users회원 정보 및 권한 관리id, username, password, email, company, role (admin/user)clients평가 대상 업체 정보id, client_name, ceo_name, industry, contactassessments업체별 평가 이력 (헤더)id, client_id, assessment_date, total_score, assessor_user_id, final_summary (AI 총평 + 수정본), is_comparisonassessment_details평가표 항목별 상세 데이터id, assessment_id, item_id, score, comment_user, comment_ai, attachment_url (첨부 파일 경로 배열)ai_settings관리자 AI 설정id, model_name, prompt_template_analysis, prompt_template_summaryreport_templates보고서 인쇄 디자인 관리id, template_name, cover_design_url, form_design_config
